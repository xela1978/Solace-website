# `font-awesome.min.css`

Derived from [Font Awesome Free 6.5.1](https://fontawesome.com) (cdnjs). Changes from upstream `all.min.css`:

- `font-display:block` → `font-display:swap` (FCP / Lighthouse)
- Webfont files are **self-hosted** under `/webfonts/` (`../webfonts/` from this file) so the critical path does not depend on cdnjs for `.woff2`.

# `site-fonts.css`

Self-hosted **Inter** + **Playfair Display** (latin + latin-ext), generated by `scripts/fetch_google_fonts_latin.py` — removes the `fonts.googleapis.com` → `fonts.gstatic.com` request chain.

Regenerate Font Awesome CSS (after updating FA version, copy new `webfonts/*.woff2` into `/webfonts/`):

```bash
python -c "import urllib.request; u='https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css'; c=urllib.request.urlopen(u).read().decode().replace('font-display:block','font-display:swap'); open('css/font-awesome.min.css','w',encoding='utf-8').write(c)"
```

Regenerate Google fonts bundle:

```bash
python scripts/fetch_google_fonts_latin.py
```
