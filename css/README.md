# `font-awesome-subset.css` (default)

Tiny stylesheet (~2 KiB) with **only the icons used on the site** (solid + brands `@font-face` + `::before` rules). Eliminates Lighthouse “unused CSS” from the full Font Awesome bundle (~21 KiB CSS).

**Adding a new icon:** look up the glyph in `font-awesome.min.css` (or [fontawesome.com](https://fontawesome.com)), then add a line to `font-awesome-subset.css`, e.g.  
`.fa-phone:before { content: "\f095"; }`  
and ensure the weight matches (solid vs brands).

# `font-awesome.min.css` (optional reference)

Full [Font Awesome Free 6.5.1](https://fontawesome.com) `all.min.css` from cdnjs — kept for lookup / diff only; **pages link `font-awesome-subset.css`**, not this file.

- `font-display:block` → `font-display:swap` in our copy
- Webfonts are **self-hosted** under `/webfonts/`

Re-download full CSS + webfonts after a FA version bump:

```bash
python -c "import urllib.request; u='https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css'; c=urllib.request.urlopen(u).read().decode().replace('font-display:block','font-display:swap'); open('css/font-awesome.min.css','w',encoding='utf-8').write(c)"
```

# `site-fonts.css`

Self-hosted **Inter** + **Playfair Display** (latin + latin-ext), generated by `scripts/fetch_google_fonts_latin.py` — removes the `fonts.googleapis.com` → `fonts.gstatic.com` request chain.

Regenerate Google fonts bundle:

```bash
python scripts/fetch_google_fonts_latin.py
```
